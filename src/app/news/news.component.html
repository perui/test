Hack for Swedens

The Hack summary:

Hack for Sweden kicked off at a racing speed with API, AI and QnA sessions. Then the hackers started coding! For us at Arbetsförmedlingen, the best part was seeing that our datasets were among the top 3 most popular APIs in use! Seven different teams used our APIs to build smart solutions for the good of the Swedish labor market. We are happy to present the seven solutions, and here they are!

The breakfast club

It's a service that suggests meetups based on the type of job the user is searching for, thus giving jobseekers the possibility to expand their professional network and increase the chance to find their dream job. This is done by combining Arbetsförmedlingen APIs (ontology and Job Forecast) and an open source API from Meetup.com Datasets: Ontology API (Arbetsförmedlingen) + Job Forecast (Arbetsförmedlingen).
Breakfast Club

Hack From Sweden

A developed “Smart Cash Index” (SCI) for plotting housing cost against remote jobs. The SCI will also show quality of life in proximity to parks, hospitals, schools etc. This is done to inform users of good places to live in while working remotely in Sweden. Datasets: Platsbanken API (Arbetsförmedlingen) + Living costs (SCB) Hack from Sweden

Ready Work

The tool provides a ranking of geographical areas where the user's unique skill-set is most needed. The algorithm takes into account the number of ads in each location and compares it with the population size in the same location.According to the 'Ready Work' team their ranking algorithm can be further developed to include the user's own preferences as a filtering method. The application also provides the user with relevant information on the region as well as a list of jobs matching their personal set of skills. The Ready Work team used Microsoft Azure’s translation API to ensure that skills are matched correctly regardless of the users language. By providing a translation service for the job application process itself, Ready Work can help connect the user with employers on a more even playing field. Datasets: Platsbanken API (Arbetsförmedlingen) + Translator API (Microsoft) Ready work

Map Your Career

Map you career facilitates the discussion on career options providing dynamic content in an easy-to-understand way. This is a tool that can be used both during on-person visits to the employment office or remotely. The newcomers can enter their skills to retrieve a list of relevant professions displayed in a bubble chart. Most relevant jobs are shown in the center while others are displayed around it. Hovering over a profession displays the job description and it's trend. Using this tool, employment officer can gather some basic background information and add their personal comments. At the end of the process a summary report is provided in a printer-friendly format. Map your career facilitates the discussion on career options providing dynamic content in an easy to understand way.p Datasets: Historical job postings, professions, Ontology and Forecasts (Arbetsförmedlingen)

Match me if you can

When selecting the topics of interest this software initiates a lookup in the graph database (Neo4J) that was initially fed with all information about the available APIs. The lookup results in a list of APIs that are related to each other and that contains data about the chosen topics. These APIs can simply be combined for building the software of your dreams. Datasets: All datasets available
Match me if you can

Match Yourself

The service matches personality with jobs. It walks you through 20 questions of a proven scientific method for understanding the jobseekers personality traits and combine the results with Arbetsförmedlingen APIs (both ontology and real-time job). The result is a matching score that suggests the most suitable jobs currently available Datasets: Ontology API + Platsbanken API (Arbetsförmedlingen)
Match yourself

LocationRank

This app produces a comparison of two geographical areas based on certain competencies helping both employers and employees to find the best geographical area for their business. Datasets: Ontology API (Arbetsförmedlingen)

Jobtech Forum

Välkommen till ett nytt Jobtech Forum den 13 oktober 2017!

Arbetsförmedlingens avdelning Digitala tjänster är glada att få bjuda in dig till höstens Jobtech Forum!

I våras genomförde vi ett välbesökt första Jobtech Forum. Nu vill vi fortsätta dialogen om hur vi gemensamt kan skapa förutsättningar för digitala samarbeten inom området jobbmatchning. Vår vision är ett ekosystem där olika aktörer delar en gemensam infrastruktur som gör det enklare för den som söker arbete eller vill rekrytera. Genom att samarbeta kan vi ta tillvara innovationskraften och bidra till en effektivare matchning på arbetsmarknaden!

Under dagen kommer vi att presentera flera nyheter bland annat en första version av Job store, Jobskills (kartläggning av kompetenser för målgruppen nyanlända) samt öppen ontologi (realtidsuppdaterat värdeföråd av kompetenser). Kan man räkna fram vilket företag som kommer att rekrytera? I samarbete mer RISE har vi tagit fram en första version av tjänsten som kombinerar historiska platsannonser med ekonomisk data.

Detta och mycket annat vill vi diskutera mer dig!


Lokal: Epicenter, Mäster Samuelsgatan 36 i Stockholm

Preliminärt program:

08.30 Registrering och kaffe

09.00 -11.00 Presentation av vision, vad som hänt sen sist och hur vi går vidare. Erik Sandström, chef för Matcha direkt, Arbetsförmedlingen

11.00-1200 Keynote speaker, Pôle Emploi, den franska statliga arbetsförmedlingen berättar om Emploi Store och deras samarbete med privata aktörer för att effektivisera matchningen på arbetsmarknaden.

12.00- 14.00 Lunch, samtidigt kan du ta del av utställningar om pågående utvecklingsinitiativ och delta i open space-diskussioner

Arbetsförmedlingen visar nya och kommande tjänster. Här kan du också diskutera affärsmodeller och samarbeten i mindre grupp. I år har vi har flera av våra skarpa utvecklare på plats för att diskutera API:er, öppna data och gemensam utveckling baserat på öppen källkod.

Dagen vänder sig till personer i ledande befattningar inom företaget och personer som arbetar med teknikutveckling. Antalet platser är begränsat.
